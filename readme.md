## 移动智能点餐系统

### 介绍
前端: Vue.js + materialize.js + JQuery.js 编写的智能点餐系统  
后端: Express.js + MongoDB + Socket.io, Django + MySQL  

### 前提
```sh
1. 打开mysql,
2. 安装依赖 npm install -d
```

### 项目运行
```sh
make run
```

### 项目调试
```sh
1. 在需要调试的地方写上debugger, 例如

>10. exports.login = function(){
>11     debugger;
>12.  }

2. make debug 断点便会显示在控制台上面。
```

### 项目测试
```sh
make test
```
